MODO ESCURO

Este é o Dark Mode ou modo escuro, uma tecnologia muito importante e muito utilizada atualmente. por exemplo, ela pode ajudar pessoas que tem algum tipo de problema visual, como fotofobia, e até mesmo pessoas que não tem problemas, mas que utilizam muito o computador para leitura, trabalho e afins, pois ela diminui o esforço que os olhos precisam fazer para poder ler a tela e, a longo prazo, evita que as pessoas desenvolvam alguma doença nos olhos. Ele pode ser feito de maneira relativamente simples com CSS/Javascript. Aqui está um video explicando uma das maneiras de se fazer o Dark Mode. Esta não é a unica forma de faze-lo, mas vale dar uma olhada, pois desenvolvedores de qualquer nivel, serão capazes de aplica-la em seus projetos, ajudando as pessoas com o modo escuro em seus sites ou aplicações web.

Dark Mode

Aqui está o código base utilizado no Darkmode, disponivel no video citado no texto acima. Você pode fazer as modificações de acordo com a sua necessidade! 
                    
                    const html = document.querySelector("html")
                    const checkbox = document.querySelector("input[name=theme]")
                    
                    const getStyle = (element, style) =>
                        window
                            .getComputedStyle(element)
                            .getPropertyValue(style)
                    
                    
                    const initialColors = {
                        bg: getStyle(html, "--bg"),
                        bgPanel: getStyle(html, "--bg-panel"),
                        colorHeadings: getStyle(html, "--color-headings"),
                        colorText: getStyle(html, "--color-text"),
                    }
                    
                    const darkMode = {
                        bg: "#333333",
                        bgPanel: "#434343",
                        colorHeadings: "#3664FF",
                        colorText: "#B5B5B5"
                    }
                    
                    const transformKey = key =>
                        "--" + key.replace(/([A-Z])/, "-$1").toLowerCase()
                    
                    
                    const changeColors = (colors) => {
                        Object.keys(colors).map(key =>
                            html.style.setProperty(transformKey(key), colors[key])
                    )
                    }
                    
                    
                    checkbox.addEventListener("change", ({target}) => {
                        target.checked ? changeColors(darkMode) : changeColors(initialColors)
                    })

ALTO CONTRASTE

A aplicação de alto contraste, é uma prática muito útil também, assim como o próprio dark mode, pois também facilita a leitura, dando mais foco no texto a ser lido em uma página, seu padrão, como pode ser visto, utilizando a tecla 'ALT + 3', nessa página, é transformar os fundos em preto e as fontes em branco, assim causando um grande contraste entre essas duas cores básicas, tornando muito mais legivel a quem tem alguma dificuldade para ler em telas de qualquer dispositivo, também tornando mais facil para o olho humano focar nos textos e outras coisas exibidas na tela.

NAVEGAÇÃO SEM MOUSE OU VIA TECLADO

A navegação sem utilizar o mouse, é uma boa prática de acessibilidade, pois auxilia as pessoas que tem algum tipo de impedimento ou baixa mobilidade, assim, a pessoa pode utilizar apenas um dispositivo/preriférico para navegar pelo seu website. Utilizamos alguns exemplos nesta página, como no inicio da página, um link para pular para o conteúdo principal, em seguida, um link para acessar o botão de dark mode, e as teclas ALT + 3 para ativar o alto contraste, como mostrado no tópico anterior, também pode ser feita a navegação entre os tópicos da página com a tecla TAB, assim facilitando para quem utiliza leitores de tela. No nosso caso, utilizamos o tabindex no HTML, pois de forma fácil ele ajuda a organizar a ordem em que a tecla TAB navega pela página, e já aplica automáticamente a borda em volta do conteudo. Existem outras maneiras de se fazer a navegação via teclado, quando necessário para sites mais complexos e com mais elementos na página. Você pode acessar um post sobre o tabindex, do desenvolvedor Everson da Luz, clicando aqui.

                        no caso usamos o ALT + 1, ativando assim o alto contraste.(Lembre-se de adicionar ao seu html
                        uma descrição explicando qual o botão deve ser pressionado para ativa-lo e também, de fazer
                        a parte funcional direto no HTML.)

                        function () {
                            var Contrast = {
                                storage: 'contrastState',
                                cssClass: 'contrast',
                                currentState: null,
                                check: checkContrast,
                                getState: getContrastState,
                                setState: setContrastState,
                                toogle: toogleContrast,
                                updateView: updateViewContrast
                        };
                        
                        window.toggleContrast = function () { Contrast.toogle(); };
                        
                        Contrast.check();
                        
                        function checkContrast() {
                            this.updateView();
                        }
                        
                        function getContrastState() {
                            return localStorage.getItem(this.storage) === 'true';
                        }
                        
                        function setContrastState(state) {
                            localStorage.setItem(this.storage, '' + state);
                            this.currentState = state;
                            this.updateView();
                        }
                        
                        function updateViewContrast() {
                            var body = document.body;
                        
                            if (!body) return;
                        
                            if (this.currentState === null)
                                this.currentState = this.getState();
                        
                            if (this.currentState)
                                body.classList.add(this.cssClass);
                            else
                                body.classList.remove(this.cssClass);
                        }
                        
                        function toogleContrast() {
                            this.setState(!this.currentState);
                        }
                        })();

BOAS PRATICAS NA DESCRIÇÃO DE IMAGENS

O texto alternativo deve corresponder à imagem e destacar de forma clara, objetiva e concisa seus pontos principais. Uma boa ideia ao criar a descrição é se perguntar "Se eu não puder usar essa imagem aqui, o que eu escrever no lugar?, porque dessa forma existe a possibilidade de que as opções de texto concordam com o que a imagem realmente transmite.As descrições das imagens podem ser notificadas no próprio texto, ou através de um atributo específico (também conhecido como ALT) do elemento img em HTML. As descrições fornecidas por meio dessa propriedade não são exibidas visualmente e so sera reconhecida por seus leitores quando o usuario navegar pelo teclado ou se ouver a falha no carregamento da imagem, dessa forma ususarios com eficiência visual podem saber qual é o conteúdo da imagem através do texto descritivo.
